# FlatFinder: Housing Revolutionized

Monorepo for the FlatFinder platform — an intelligent housing search system built around the 40% affordability rule.

## Packages

| Package | Description |
|---------|-------------|
| `database` | Supabase schemas and migrations |
| `scraper` | Listing aggregation system |
| `algorithm` | 40% affordability engine |
| `web` | Next.js dashboard application |
| `chatbot` | AI assistant implementation |
| `deployment` | CI/CD and infrastructure |
| `testing` | Test suites and QA tools |
| `docs` | Documentation and pitch materials |

## Getting Started

```bash
npm install
npx turbo dev
```

## Local Development

```bash
docker compose up -d
```
